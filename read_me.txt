Создание проекта idea
   Делаем пункты описанные в файле idea.gradle
     при этом конфигурацию запуска можно создать и в ручную
   Запускаем idea и открываем сгенерированный файл *.ipr
   В EventLog появятся сообщения, нужно выполнить "Import Gradle project" и др.
   Теперь можно запускать конфигурацию "GlassFish", если она была добавлена по инструкции выше, или запускать другую созданную конфигурацию

Для запуска и развертывания web проекта на GlassFish сервер, надо актуализировать переменную "glassfishHome" в файле gradle.properties,
    запусктить таск "runDeploy", при этом сервер должен быть уже стартован

Для запуска модульных тестов тока надо определится как это делается для web, но на данный момент уже можно запускать таск "build", 
        он выполнит компиляцию и прогонку unit tests вне контекста web сервера

AspectJ это более широкий набор функциональности, который можно использовать если не хватает аспектов Spring
АОП может быть статическим - на этапе компиляции, и динамическим.

Терминология
    Точки соединения (joinpoint) - это четко определенная точка - вызов метода, инициализация класса, создание объекта
    Советы (advice) - это код, который должен выполнится в точке соединения, совет может выполняться перед и\или после
        точки соединения.
    Срез (pointcut) - коллекция точек соединения, к которым применяется совет.
    Аспект (aspect) - это комбинация совета и срезов, которая определяет логику, которая должна быть включена в приложение.
    Связывание (weaving) - это процесс вставки асректа в определенную точку, во время компиляции, выполнения или загрузуки
    Цель (target) - объект, поток выполнения которонго меняется какимто процессом АОП
    Введение - (introdaction) процесс позволяющий изменить структуру объекта за счет помещения в него дополнительных методов и полей,
        АОП позаоляет сделать реализацию объектом интерфейса без необходимости реализации классом этого интерфейса явно.
